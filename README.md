## Getting Started

Clone project and running test in local  
```
http://localhost:4200
```

### Prerequisites

You need to install the software.

- Angular version 5 or greater

Install global angular cli

```
npm install -g @angular/cli@latest

// If error install

npm i -g npm
npm cache verify
npm cache clean --force

// Already again

npm install -g @angular/cli@latest

```
